Medi‑Solutions AI Agent Suite
A working, click‑through prototype you can host on GitHub Pages today.
What’s included
  • Dashboard: KPIs, today’s focus, follow‑up templates
  • Content Engine: 1 topic → scripts/posts/email/blog
  • Sales Coach: objection lab + roleplay scoring
  • Client Tools: IRMAA explainers + employer check
  • Snapshot Builder: intake → snapshot → save to Lead Inbox
  • Lead Inbox: status updates + next actions
  • Library: scripts, compliance phrases, playbooks
  • Admin: weekly theme + settings + approvals queue
Enter Suite
Quick start
This prototype stores data in your browser (localStorage). No backend.
Recommended flow demo
  1. Open Snapshot Builder and generate a snapshot.
  2. Click Save to Lead Inbox.
  3. Go to Lead Inbox → open the lead → update status.
  4. Go to Content Engine → generate a multi‑platform kit.
  5. (Admin role) submit a draft → approve it in Admin.

Where to customize

Brand + Compliance toggles are in the top bar.
